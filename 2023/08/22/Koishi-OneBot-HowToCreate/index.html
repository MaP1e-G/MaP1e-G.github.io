问题

获取T544 sign时出现错误: encoding/hex: invalid byte: U+0069 ‘i’

使用1.1.0版本的unidbg-fetch-qsign，或者使用go-cqhttp的dev分支（需要自行编译）。
参考来源：[Bug]: 获取T544 sign时出现错误: encoding/hex: invalid byte: U+0069 ‘i’ #2304

参考gocq的issue里边使用docker搭建qsign服务跑不起来

用docker container logs qsign看看是不是没填android_id，是的话加个参数：docker run -d --restart=always --name qsign -p 8080:8080 -e ANDROID_ID=3e9e151c8b55af91 xzhouqd/qsign:8.9.63，记得替换成自己的device.json（这个文件就在go-cqhttp根目录下）里面的android_id。
参考来源：签名服务器相关问题 #2242
