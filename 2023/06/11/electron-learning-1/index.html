要把页面加载进应用窗口中，需要两个electron模块：

  • app模块，控制应用程序的事件生命周期
  • BrowserWindow模块，创建和管理应用程序窗口
    在main.js文件头部将它们导入作为CommonJS模块：const { app, BrowserWindow } = require('electron')
